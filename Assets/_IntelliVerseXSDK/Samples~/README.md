# IntelliVerseX SDK Samples

This folder contains sample projects demonstrating how to use the IntelliVerseX SDK.

## Importing Samples

Samples can be imported through Unity's Package Manager:

1. Open **Window > Package Manager**
2. Find **IntelliVerseX SDK** in the list
3. Expand the **Samples** section
4. Click **Import** next to the sample you want

## Available Samples

### Quiz Demo
A complete quiz game implementation showing:
- Quiz session management
- Question shuffling
- Score tracking
- Leaderboard integration

### Localization
Multi-language UI demonstration with:
- 12+ language support
- RTL (Right-to-Left) layout support
- Dynamic language switching
- Localized asset loading

### IAP Integration
In-app purchase implementation showing:
- Product configuration
- Purchase flow
- Receipt validation
- Subscription handling

### Leaderboard
Global leaderboard example with:
- Nakama backend integration
- Score submission
- Ranking display
- Around-player rankings

### Social Features
Social system demonstration with:
- Friends system
- Native sharing
- Referral system
- User profiles

## Sample Structure

Each sample follows this structure:

```
SampleName/
├── Scenes/           # Demo scenes
├── Scripts/          # Example scripts
├── Prefabs/          # UI prefabs
├── Resources/        # Sample resources
└── README.md         # Sample documentation
```

## Note

These samples require the IntelliVerseX SDK and its dependencies to be properly installed.
See the main SDK documentation for dependency installation instructions.
