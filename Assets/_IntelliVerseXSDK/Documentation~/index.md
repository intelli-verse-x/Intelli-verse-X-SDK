## SDK Overview

IntelliVerseX SDK is a comprehensive, modular game development toolkit for Unity that provides:

### Core Features
- **Identity & Authentication**: Multi-provider auth (Device, Email, Apple, Google)
- **Backend Integration**: Nakama server connectivity out of the box
- **Monetization**: IAP, Ads (LevelPlay, Appodeal, AdMob), Offerwalls
- **Analytics**: Event tracking and user behavior analysis
- **Localization**: 12+ languages with RTL support
- **Storage**: Secure cloud saves and local persistence
- **Leaderboards**: Global rankings with Nakama
- **Social**: Friends, sharing, referrals
- **Quiz System**: Complete quiz game framework
- **UI Components**: Production-ready UI utilities

### Supported Platforms
- Android (API 21+)
- iOS (12+)
- WebGL
- Windows Standalone
- macOS Standalone

### Unity Version Support
- **Minimum**: Unity 2023.3 LTS
- **Recommended**: Unity 6000.x
- **Tested On**: Unity 2023.3, Unity 6000.0
