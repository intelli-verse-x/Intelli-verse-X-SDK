# 📊 Tracking Documentation

This folder contains tracking documentation for the IntelliVerseX Unity SDK.

## Files

| File | Purpose |
|------|---------|
| `README.md` | This file |
| `BUGS.md` | Bug tracking (when created) |
| `TASKS.md` | Task tracking (when created) |

## Bug Tracking

When bugs are discovered, they should be documented in `BUGS.md` with:

- Bug ID
- Description
- Severity
- Status
- Steps to reproduce
- Resolution

## Task Tracking

For larger tasks, use `TASKS.md` to track:

- Task ID
- Description
- Priority
- Status
- Assignee
- Due date
