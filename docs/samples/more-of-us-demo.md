# More Of Us Demo

This sample shows how to implement the "More Of Us" cross-promotion feature.

## Scene

`IVX_MoreOfUs.unity`

## Features

- Cross-promote other games
- Deep linking support
- Install tracking
- Reward incentives

## Setup

1. Configure games in `IVXGameConfig`
2. Add `IVXMoreOfUsManager` to scene
3. Display cross-promotion UI
