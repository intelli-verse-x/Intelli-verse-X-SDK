# Weekly Quiz Demo

This sample demonstrates implementing weekly quiz tournaments.

## Scene

`IVX_WeeklyQuizTest.unity`

## Features

- Weekly quiz competitions
- Global leaderboards
- Prize distribution
- Tournament brackets

## Setup

1. Import the sample from Package Manager
2. Configure weekly quiz in backend
3. Add UI prefabs to your scene
