# Support

Need help with IntelliVerseX SDK? Here's how to get support.

## Supported Platforms

| Platform | SDK Location |
|----------|-------------|
| Unity / .NET | `Assets/_IntelliVerseXSDK/` |
| Unreal Engine | `SDKs/unreal/` |
| Godot Engine | `SDKs/godot/` |
| Defold | `SDKs/defold/` |
| Cocos2d-x | `SDKs/cocos2dx/` |
| JavaScript | `SDKs/javascript/` |
| C / C++ | `SDKs/cpp/` |
| Java / Android | `SDKs/java/` |

## 🐛 Bug Reports

Found a bug? Report it on GitHub:

[Create Bug Report](https://github.com/AhamedAzmi/IntelliVerseX/issues/new?template=bug_report.md)

**Include:**
- Unity version
- SDK version
- Platform (Android/iOS/WebGL/etc.)
- Steps to reproduce
- Expected vs actual behavior
- Error messages/logs

## 📧 Email Support

For private inquiries or enterprise support:

**General Support:** support@intelli-verse-x.ai

**Security Issues:** security@intelli-verse-x.ai (Do NOT file public issues for security vulnerabilities)

**Response Time:**
- Community support: Best effort
- Email support: 24-48 hours

---

## 🏢 Enterprise Support

For enterprise customers requiring:
- Priority support
- SLA guarantees
- Custom integrations
- On-premise deployment

Contact: enterprise@intelli-verse-x.ai
